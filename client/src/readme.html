Retro Store

Second hand clothes

Additonal pages:
  • WishList - a part of the menu. In case you want to add a cloth to the wish list, in order to remember this item, you click on the heart symbol next to it. Then you can see your list under wish list tab.
  • Contact us - a part of the menu. Generates an email with your request.
  • Filtered clothes pages: Tops, Pants, Skirts, Dresses - sub menu of products. Filter clothes by category, if you click a category you can see all the cloth in this category.

How did you make your store secured?

We checked the input parameters, we used try and catch block in places where exceptions might be thrown.
When a user logs in with a correct password we generate a cookie with a unique id without saving in the session the user's details.
